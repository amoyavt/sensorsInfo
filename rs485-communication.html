RS485 Communication

Request: Query the version number

address code function code Register address high byte Register address low byte High byte of the number of registers Low byte of register number CRC check
0xXX 0x03 0x00 0x24 0x00 0x06 High byte, low byte

Response

address code function code Return the number of bytes High byte of valid data CRC check
0xXX 0x03 0x0C (This machine version number) High byte, low byte

Example

Send: 01 03 00 24 00 06 85 C3
Return: 01 03 0C 32 30 32 30 36 32 38 30 31 30 31 5B 18 28 01
(2020 06 28 01 01)
